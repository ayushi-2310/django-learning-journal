#  Django Learning Journal

Welcome to my **Django Learning Journal** — a collection of notes, code snippets, and mini-projects documenting my journey of learning the Django web framework.

---

##  Structure

| Folder | Description |
|--------|--------------|
| `00-introduction/` | Basics and setup |
| `01-core-concepts/` | Models, Views, Templates, URLs, etc. |
| `02-advanced-topics/` | Signals, Middleware, Authentication |
| `03-projects/` | Small Django practice projects |
| `resources.md` | Helpful tutorials, docs, and links |

---

##  Goals

- Master Django step by step  
- Build small projects for each concept  
- Maintain clean, markdown-based notes  

---

##  Learning Path

1. **Introduction** → What Django is, installation, and first app  
2. **Core Concepts** → Deep dive into MTV pattern, ORM, and views  
3. **Advanced Topics** → Class-based views, signals, middleware  
4. **Projects** → Build mini Django apps (To-do list, Blog, etc.)
